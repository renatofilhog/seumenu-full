# Feature: Order Tracking

## What
Customers and restaurant staff can follow order progress through defined status stages from submission to completion.

## Why
Visible order progress improves transparency for customers and helps staff coordinate kitchen and floor operations more effectively.

## Acceptance Criteria
- [ ] Customers can see the latest status for a recent order
- [ ] Staff can move orders between operational stages
- [ ] Status changes are reflected in the operational workflow
